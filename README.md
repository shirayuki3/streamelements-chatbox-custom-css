# streamelements-chatbox
OBS側で使用するカスタムCSSです。適宜OBS側でクロップする必要があります。自分専用に作成したため、以下のStreamElementsの設定でのみ正常に動作します。カスタマイズする際はCSSをご自身で修正していただく必要がありますのでご了承ください。
